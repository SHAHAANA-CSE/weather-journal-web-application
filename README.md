# Weather Journal Web Application

## Overview
Weather Journal is a responsive web application developed using HTML, CSS, and JavaScript that provides real-time weather information for any city using the OpenWeatherMap API. The application displays temperature, weather conditions, humidity, wind speed, and dynamic background images based on the searched location.

## Features
- Search weather information by city name
- Real-time weather data using OpenWeatherMap API
- Displays temperature, humidity, and wind speed
- Dynamic weather icons and background images
- Responsive and user-friendly interface
- Error handling for invalid city names

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## Project Structure
- `index.html` – Defines the structure and layout of the application.
- `style.css` – Provides styling and responsive user interface design.
- `script.js` – Handles API integration and application functionality.

## How to Run
1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Enter a city name and click **Check** to view the current weather details.
